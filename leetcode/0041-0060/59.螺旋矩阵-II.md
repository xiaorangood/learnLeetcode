# 题目描述

这是LeetCode上的[59.螺旋矩阵-II](https://leetcode.cn/problems/spiral-matrix-ii/)。

难度：**中等**

Tag：数组、矩阵、模拟

给你一个正整数`n`，生成一个包含`1`到`n2`所有元素，且元素按顺时针顺序螺旋排列的`n x n`正方形矩阵`matrix`。

示例1：

```
    1 -> 2 -> 3
              ↓ 
    8 -> 9    4
    ↑         ↓
    7 <- 6 <- 5
输入：n = 3
输出：[[1,2,3],[8,9,4],[7,6,5]]
```


示例2：

```
输入：n = 1
输出：[[1]]
```

提示：

- `1 <= n <= 20`

# 解题思路

- 生成一个`n×n`空矩阵`mat`，随后模拟整个向内环绕的填入过程：
  - 定义当前左右上下边界`l`、`r`、`t`、`b`，初始值`num = 1`，迭代终止值`tar = n * n`；
  - 当`num <= tar`时，始终按照"**从左到右**"、“**从上到下**”、“**从右到左**”、“**从下到上**”填入顺序循环，每次填入后：
    - 执行`num += 1`：得到下一个需要填入的数字；
    - 更新边界：例如从左到右填完后，上边界`t += 1`，相当于上边界向内缩1。
  - 使用`num <= tar`而不是`l < r || t < b`作为迭代条件，是为了解决当`n`为奇数时，矩阵中心数字无法在迭代过程中被填充的问题。
- 最终返回`mat`即可。

## 代码

```c++
class Solution {
public:
    vector<vector<int>> generateMatrix(int n) {
        int l = 0;
        int r = n-1;
        int t = 0;
        int b = n-1;
        vector<vector<int>> mat(n,vector<int>(n, 0));
        int num = 1, tar = n * n;
        while(num <= tar){
            for(int i = l; i <= r; i++)
                mat[t][i] = num++;
            t++;
            for(int i = t; i <= b; i++)
                mat[i][r] = num++;
            r--;
            for(int i = r; i >= l; i--)
                mat[b][i] = num++;
            b--;
            for(int i = b; i >= t; i--)
                mat[i][l] = num++;
            l++;
        }
        return mat;
    }
};
```

## 复杂度分析

- **时间复杂度**$O(N)$：`N`分别为矩阵元素个数。
- **空间复杂度**$O(1)$：四个边界`l`、`r`、`t`、`b`使用常数大小的额外空间。
