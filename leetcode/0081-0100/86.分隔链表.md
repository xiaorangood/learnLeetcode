# 题目描述

这是LeetCode上的[86.分隔链表](https://leetcode.cn/problems/partition-list/)。

难度：**中等**

Tag：链表、双指针

给你一个链表的头节点`head`和一个特定值`x`，请你对链表进行分隔，使得所有**小于**`x`的节点都出现在**大于或等于**`x`的节点之前。

你应当**保留**两个分区中每个节点的初始相对位置。

示例1：

```
输入：head = [1,4,3,2,5,2], x = 3
输出：[1,2,2,4,3,5]
```


示例2：

```
输入：head = [2,1], x = 2
输出：[1,2]
```

# 解题思路

## 算法流程

根据题意，考虑通过**新建两个链表**实现原链表分割，算法流程为：

- 新建两个链表`dummySmall`，`dummyBig`，分别用于添加所有`节点值 < x`、`节点值 ≥ x`的节点。
- 遍历链表`head`并依次比较各节点值`head.val`和`x`的大小：
  - 若`head.val < x`，则将节点`head`添加至链表`dummySmall`最后面；
  - 若`head.val >= x`，则将节点`head`添加至链表`dummyBig`最后面；
- 遍历完成后，拼接`dummySmall`和`dummyBig`链表。
- 最终返回头节点`dummySmall.next`即可。

## 代码

```c++
class Solution {
public:
    ListNode* partition(ListNode* head, int x) {
        // 新建两个链表
        ListNode dummySmall(0);
        ListNode dummyBig(0);
        // 遍历链表
        ListNode* ds = &dummySmall
        ListNode* db = &dummyBig;
        while(head != nullptr){
            if(head->val < x){
                // 将 < x 的节点加入 ds 节点后
                ds->next = head;
                ds = ds->next;
            }
            else{
                // 将 < x 的节点加入 db 节点后
                db->next = head;
                db = db->next;
            }
            head = head->next;
        }
        // 拼接两链表
        ds->next = dummyBig.next;
        db->next = nullptr;
        return dummySmall.next;
    }
};
```

## 复杂度分析

- **时间复杂度$O(N) $：** $N$为链表长度；遍历链表使用线性时间。
- **空间复杂度$O(1)$：** 假头节点使用常数大小的额外空间。
