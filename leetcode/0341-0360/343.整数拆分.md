# 题目描述

这是LeetCode上的[343.整数拆分](https://leetcode.cn/problems/integer-break/)。

难度：**中等**

Tag：数学、动态规划

给定一个正整数`n`，将其拆分为`k`个**正整数**的和（`k >= 2`），并使这些整数的乘积最大化。

返回你可以获得的最大乘积。

示例1：

```
输入: n = 2
输出: 1
解释: 2 = 1 + 1, 1 × 1 = 1。
```


示例2：

```
输入: n = 10
输出: 36
解释: 10 = 3 + 3 + 4, 3 × 3 × 4 = 36。
```

提示：

- `2 <= n <= 58`

# 解题思路

设将整数`n`拆分为`a`个小数字： $n=n_1+n_2+\dots+n_a$

本题等价于求解： $\max(n_1 \times n_2 \times \dots \times n_a)$

**推论一**：若拆分的数量`a`确定，则**各拆分数字相等时**，乘积最大。

- 以下公式为“算术几何均值不等式”，等号当且仅当 $n_1=n_2=\dots=n_a$时成立。

$$
\frac{n_1 + n_2 + \ldots + n_a}{a} \geq \sqrt[a]{n_1 n_2 \ldots n_a}
$$

**推论二**：将数字`n`尽可能以因子`3`等分时，乘积最大。

- 设将数字以因子`x`等分为`a`个，即 $n=ax$，则乘积为 $x^a$。

- 观察 $x^a = x^{\frac{n}{x}} = (x^{\frac{1}{x}})^n$，由于`n`为常数，因此当 $x^\frac{1}{x}$取最大值时，乘积达到最大值。

- 根据分析，可将问题转化为求 $y = x^{\frac{1}{x}}$的极大值。

  - 先两边取自然对数： $\ln y = \frac{1}{x} \ln x$。
  - 左边对`x`求导： $\frac{d}{dx}(\ln y) = \frac{1}{y} \cdot \frac{dy}{dx}$
  - 右边对`x`求导： $\frac{d}{dx} \left( \frac{1}{x} \ln x \right) = \left( \frac{d}{dx} \frac{1}{x} \right) \ln x + \frac{1}{x} \cdot \frac{d}{dx} (\ln x) =  -\frac{1}{x^2} \ln x + \frac{1}{x} \cdot \frac{1}{x} = \frac{1 - \ln x}{x^2}$
  - 故对`x`求导后得到： $\frac{1}{y} \cdot \frac{dy}{dx} = \frac{1 - \ln x}{x^2}$
  - 两边同时乘以`y`，并将 $y=x^\frac{1}{x}$带入，得： $\frac{dy}{dx} = x^{\frac{1}{x}} \cdot \frac{1 - \ln x}{x^2}$
  - 令导数为`0`求得极大值，即 $x^{\frac{1}{x}} \cdot \frac{1 - \ln x}{x^2}=0$。
  - 由于当`x > 0`时， $x^\frac{1}{1} > 0$，所以只需分子为零，即 $1-\ln x = 0$可得到极大值。
  - 因此函数在 $x = e \approx 2.7$处取得极大值。
  - 由于因子`x`必须为整数，最接近`e`的整数为2或3。
  - 如下式所示，代入 $x = 2$ 和 $x = 3$，得出 $x = 3$时，乘积达到最大。

  $$
  \begin{aligned}
  y(3) &= 3^{1/3} \approx 1.44 \\
  y(2) &= 2^{1/2} \approx 1.41
  \end{aligned}
  $$


## 拆分规则

- **最优**：`3`。把数字`n`可能拆为多个因子`3`，余数可能为`0`，`1`，`2`三种情况。
- **次优**：`2`。若余数为`2`；则保留，不再拆为`1 + 1`。
- **最差**：`1`。若余数为`1`；则应把一份`3 + 1`替换为`2 + 2`，因为`2 * 2 > 3 * 1`。

## 算法流程

- 当`n <= 3`时，按照规则应不拆分，但由于题目要求必须拆分，因此必须拆出一个因子`1`，即返回`n−1`。
- 当`n > 3`时，求`n`除以`3`的整数部分`a`和余数部分`b`（即`n = 3a + b`），并分为以下三种情况：
  - 当`b = 0`时，直接返回 $3^a$；
  - 当`b = 1`时，要将一个`1 + 3`转换为`2 + 2`，因此返回 $3^{a−1} \times 4$；
  - 当`b = 2`时，返回 $3^a \times 2$。

## 代码

```c++
class Solution {
public:
    int integerBreak(int n) {
        if(n <= 3) return n - 1;
        int a = n / 3, b = n % 3;
        if(b == 0) return pow(3, a);
        if(b == 1) return pow(3, a - 1) * 4;
        return pow(3, a) * 2;
    }
};
```

- **时间复杂度**$O(1)$：仅有求整、求余、次方运算。
  - [求整和求余运算](https://stackoverflow.com/questions/35189851/time-complexity-of-modulo-operator-in-python)：查阅资料，提到不超过机器数的整数可以看作是 $O(1)$；
  - [幂运算](https://stackoverflow.com/questions/32418731/java-math-powa-b-time-complexity)：查阅资料，提到浮点取幂为 $O(1)$。
- **空间复杂度**$O(1)$：`a`和`b`使用常数大小额外空间。
