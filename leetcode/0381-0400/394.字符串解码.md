# 题目描述

这是LeetCode上的[394.字符串解码](https://leetcode.cn/problems/decode-string/)。

难度：**中等**

Tag：栈、递归、字符串

给定一个经过编码的字符串，返回它解码后的字符串。

编码规则为：`k[encoded_string]`，表示其中方括号内部的`encoded_string`正好重复`k`次。注意`k`保证为正整数。

你可以认为输入字符串总是有效的；输入字符串中没有额外的空格，且输入的方括号总是符合格式要求的。

此外，你可以认为原始数据不包含数字，所有的数字只表示重复的次数`k`，例如不会出现像`3a`或`2[4]`的输入。

测试用例保证输出的长度不会超过`105`。

示例1：

```
输入：s = "3[a]2[bc]"
输出："aaabcbc"
```


示例2：

```
输入：s = "3[a2[c]]"
输出："accaccacc"
```

示例3：

```
输入：s = "2[abc]3[cd]ef"
输出："abcabccdcdcdef"
```

示例4：

```
输入：s = "abc3[cd]xyz"
输出："abccdcdcdxyz"
```

提示：

- `1 <= s.length <= 30`
- `s`由小写英文字母、数字和方括号`'[]'`组成
- `s`保证是一个**有效**的输入。
- `s`中所有整数的取值范围为`[1, 300]`

# 解题思路

## 方法一：辅助栈法

本题难点在于括号内嵌套括号，需要从内向外生成与拼接字符串，这与栈的先入后出特性对应。

### 算法流程

- 构建辅助栈`stk_multi`和`stk_str`，遍历字符串`s`中每个字符`c`；
  - 当`c`为数字时，将数字字符转化为数字`multi`，用于后续倍数计算；
  - 当`c`为字母时，在`res`尾部添加`c`；
  - 当`c`为`[`时，将当前`multi`和`res`分别入栈`stk_multi`和`stk_str`，并分别清空`multi`和`res`：
    - 记录此`[`前的临时结果`res`至栈，用于发现对应`]`后，进行拼接操作；
    - 记录此`[`前的倍数`multi`至栈，用于发现对应`]`后，获取`multi × [...]`字符串。
    - 进入到新`[`后，`res`和`multi`重新记录。
  - 当`c`为`]`时，`stk_str`出栈并复制给`last_res`，拼接字符串`res = last_res + cur_multi * res`，其中:
    - `last_res`是上个`[`到当前`[`的字符串，例如"3[a2[c]]"中的a；
    - `cur_multi`是当前`[`到`]`内字符串的重复倍数，例如"3[a2[c]]"中的2。
- 返回字符串`res`。

### 代码

```c++
class Solution {
public:
    string decodeString(string s) {
        vector<string> stk_str;
        vector<int> stk_multi;
        int multi = 0;
        string res;
        for (auto ch:s){
            if(ch=='['){
                // multi 和 res 入栈
                stk_multi.push_back(multi);
                stk_str.push_back(res);
                // multi 和 res 清空
                multi = 0;
                res.clear();
            }
            else if(ch == ']'){
                string temp;
                // 获取当前字符串的重复次数
                int cur_multi = stk_multi.back();
                stk_multi.pop_back();
                // 生成重复的字符串
                for(int i = 0; i < cur_multi; i++){
                    temp += res;
                }
                // 将重复字符串和倍数之前的字符串拼接
                res = stk_str.back() + temp;
                stk_str.pop_back();
            }
            else if(ch >= '0' && ch <= '9'){
                multi = multi * 10 + (ch - '0');
            }
            else{
                res += ch;
            }
        }
        return res;
    }
};
```

### 复杂度分析

- **时间复杂度**$O(N)$：一次遍历`s`；
- **空间复杂度**$O(N)$：辅助栈在极端情况下需要线性空间，例如`2[2[2[a]]]`。

## 方法二：递归法

总体思路与辅助栈法一致，不同点在于将`[`和`]`分别作为递归的开启与终止条件：

- 当`s[i] == ']'`时，返回当前括号内记录的`res`字符串与`]`的索引`i`（更新上层递归指针位置）；
- 当`s[i] == '['`时，开启新一层递归，记录此`[...]`内字符串`tmp`和递归后的最新索引`i`，并执行`res + multi * tmp`拼接字符串。
- 遍历完毕后返回`res`。

### 代码

```c++
class Solution {
public:
    string decodeString(string s) {
        int index = 0;
        return dfs(s,index);
    }
private:
    string dfs(const string& s, int& i){
        string res;
        int multi = 0;
        while(i < s.length()) {
            char ch = s[i];
            if(isdigit(ch)){
                // 解析数字
                multi = multi * 10 + (ch - '0');
            }
            else if (ch == '[') {
                // 遇到 '['，递归解码内部字符串
                i++;  // 跳过 '['
                string inner = dfs(s, i);
                // 将内部字符串重复 multi 次
                for (int k = 0; k < multi; k++) {
                    res += inner;
                }
                multi = 0;  // 重置数字
            } else if (ch == ']') {
                // 遇到 ']'，返回当前层的结果
                return res;
            } else {
                // 普通字符，直接添加
                res += ch;
            }
            i++;
        }
        return res;
    }
};
```

- **时间复杂度**$O(N)$，递归会更新索引，因此实际上还是一次遍历`s`；
- **空间复杂度**$O(N)$，极端情况下递归深度将会达到线性级别
