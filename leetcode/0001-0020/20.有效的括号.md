# 题目描述

这是LeetCode上的[20.有效的括号](https://leetcode.cn/problems/valid-parentheses/)。

难度：**简单**

Tag：栈、字符串

给定一个只包括`'('`，`')'`，`'{'`，`'}'`，`'['`，`']'`的字符串`s`，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

示例1：

```
输入：s = "()"
输出：true
```


示例2：

```
输入：s = "()[]{}"
输出：true
```

示例3：

```
输入：s = "(]"
输出：false
```

示例4：

```
输入：s = "([])"
输出：true
```

示例5：

```
输入：s = "([)]"
输出：false
```

提示：

- `1 <= s.length <= 104`
- `s`仅由括号`'()[]{}'`组成

# 解题思路

## 算法原理

- 若遇到左括号入栈，遇到右括号时将对应栈顶左括号出栈，则遍历完所有括号后`stack`仍然为空。
- 建立哈希表`dic`构建左右括号对应关系，这样查询2个括号是否对应只需$O(1)$时间复杂度。

## 算法流程

- 如果`c`是左括号，则入栈；
- 否则通过哈希表判断括号对应关系，若`stack`栈顶出栈括号`stack.pop()`与当前遍历括号`c`不对应，则提前返回false。

## 提前返回方法

- 栈`stack`为空。
- 字符串`s`以左括号开始。

## 代码

```c++
class Solution {
public:
    bool isValid(string s) {
        unordered_map<char,char> map = {
            {')', '('},
            {']', '['},
            {'}', '{'}
        };
        // String括号成对，奇数表示不成对
        int len = s.size();
        if (len % 2 == 1 || (len > 0 && map.count(s[0]))){
            return false;
        }
        stack<char> stk;
        for(auto c : s){
            if(map.count(c)){
                if(stk.empty() || stk.top() != map[c])
                {
                    return false;
                }
                stk.pop();
            }
            else{
                stk.push(c);
            }
        }
        return stk.empty();
    }
};
```

## 复杂度分析

- **时间复杂度**$O(n)$：，其中`n`是字符串`s`的长度。
- **空间复杂度**$O(n+\left| \Sigma \right|)$：，其中$\Sigma$表示字符集，本题中字符串只包含6种括号，$\left| \Sigma \right| = 6$。栈中的字符数量为$O(n)$，而哈希表使用的空间为$O(\left| \Sigma \right|)$，相加即可得到总空间复杂度。
