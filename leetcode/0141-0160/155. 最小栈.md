# 题目描述

这是LeetCode上的[155. 最小栈](https://leetcode.cn/problems/min-stack/)。

难度：**中等**

Tag：栈、设计

设计一个支持`push`，`pop`，`top`操作，并能在常数时间内检索到最小元素的栈。

实现`MinStack`类:

- `MinStack()`初始化堆栈对象。
- `void push(int val)`将元素val推入堆栈。
- `void pop()`删除堆栈顶部的元素。
- `int top()`获取堆栈顶部的元素。
- `int getMin()`获取堆栈中的最小元素。

示例1：

```
输入：
["MinStack","push","push","push","getMin","pop","top","getMin"]
[[],[-2],[0],[-3],[],[],[],[]]

输出：
[null,null,null,null,-3,null,0,-2]

解释：
MinStack minStack = new MinStack();
minStack.push(-2);
minStack.push(0);
minStack.push(-3);
minStack.getMin();   --> 返回 -3.
minStack.pop();
minStack.top();      --> 返回 0.
minStack.getMin();   --> 返回 -2.
```

提示：

- `-231 <= val <= 231 - 1`
- `pop`、`top`和`getMin`操作总是在**非空栈**上调用
- `push`、`pop`、`top`和`getMin`最多被调用`3 * 104`次

# 解题思路

借用一个辅助栈`min_stk`，用于存获取`stk`中最小值。

## 算法流程

- **`push()` 方法：**每当新值`val`进来时，`stk`对象直接在`vector`尾添加数值，`min_stack`对象将`val`和`min_stack`栈顶值添加到`vector`的末尾，长度与`stack`一样；
- **`pop()` 方法：** 两个栈都使用pop`()`方法。
- **`getMin()`方法：** 返回 `min_stack`对象最后一个元素即可。

## 代码

```c++
class MinStack {
private:
    vector<int> stk;
    vector<int> min_stk;
public:
    MinStack() {
        
    }
    
    void push(int val) {
        stk.push_back(val);
        if(min_stk.empty()){
            min_stk.push_back(val);
        }else{
            min_stk.push_back(std::min(val, min_stk.back()));
        }
    }
    
    void pop() {
        stk.pop_back();
        min_stk.pop_back();
    }
    
    int top() {
        return stk.back();
    }
    
    int getMin() {
        return min_stk.back();
    }
};
```

## 复杂度分析

- **时间复杂度**$O(n)$：，其中`n`是字符串`s`的长度。
- **空间复杂度**$O(n+\left| \Sigma \right|)$：，其中$\Sigma$表示字符集，本题中字符串只包含6种括号，$\left| \Sigma \right| = 6$。栈中的字符数量为$O(n)$，而哈希表使用的空间为$O(\left| \Sigma \right|)$，相加即可得到总空间复杂度。
